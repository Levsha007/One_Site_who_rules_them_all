# My Site

Полнофункциональный веб-сайт для управления изображениями, закладками и профилем GitHub с современным дизайном и интерактивными возможностями.

---

## Особенности

### Галерея изображений

- Masonry-сетка с адаптивным количеством колонок (1–10)
- Загрузка файлов и URL изображений
- Создание папок/галерей для организации
- Модальный просмотр с навигацией стрелками
- Локальные избранные и переименования
- Пагинация (150 изображений на страницу)
- Drag & Drop и вставка из буфера обмена (Ctrl+V)
- Долгое нажатие для быстрого перехода в папку

### Система закладок

- Два режима просмотра: сетка и список
- Категории: Работа, Образование, Игры, Новости, Развлечения, Социальные сети, Спорт, Технологии
- Поиск и сортировка по названию и категории
- Автоматические фавиконы через Google API
- CRUD-операции (создание, чтение, обновление, удаление)

### GitHub Профиль

- Получение данных через GitHub API
- Просмотр репозиториев, избранного и подписок
- Встроенный просмотр файлов (Markdown, код, изображения)
- Base64-декодирование с поддержкой кириллицы
- Навигация по структуре репозиториев

### Параллакс-страница

- 3D эффекты параллакса при скролле
- Плавная анимация с GSAP и ScrollSmoother
- Слоистая композиция с лесной тематикой

### Дизайн-система

- Темная/светлая тема с автоматическим переключением
- CSS-переменные для единообразия стилей
- Адаптивный дизайн под все устройства
- Кастомные шрифты: *Brush Script MT*, *Playfair Display*, *Poppins*, *Raleway*

## Технологии

### Frontend

- **HTML5** — семантическая разметка
- **CSS3** — Grid, Flexbox, CSS-переменные, анимации
- **Vanilla JavaScript** — без фреймворков
- **Masonry.js** — адаптивная сетка изображений
- **GSAP + ScrollSmoother** — параллакс-анимации

### Backend

- **Node.js + Express** — серверная платформа
- **Multer** — обработка загрузок
- **Sharp** — обработка и оптимизация изображений
- **REST API** — структурированные эндпоинты

### Хранение данных

- **JSON-файлы** — галереи, закладки, категории
- **LocalStorage** — пользовательские предпочтения
- **Файловая система** — загруженные изображения

## Установка и запуск

### Предварительные требования

- Node.js 14+
- npm или yarn

### Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd my-gallery

# Установка зависимостей
npm install

# Запуск сервера разработки
npm start

# Или с автоматическим перезапуском
npm run dev
```

Сервер будет доступен по адресу:\
[**http://localhost:3000**](http://localhost:3000)

## API Эндпоинты

### Галереи

| Метод | Эндпоинт          | Описание                  |
| ----- | ----------------- | ------------------------- |
| GET   | `/galleries`      | Получить все галереи      |
| POST  | `/create-gallery` | Создать новую галерею     |
| POST  | `/upload`         | Загрузить изображения     |
| POST  | `/rename`         | Переименовать изображение |
| POST  | `/delete-image`   | Удалить изображение       |

### Закладки

| Метод  | Эндпоинт         | Описание              |
| ------ | ---------------- | --------------------- |
| GET    | `/bookmarks`     | Получить все закладки |
| POST   | `/bookmarks`     | Создать закладку      |
| DELETE | `/bookmarks/:id` | Удалить закладку      |
| GET    | `/categories`    | Получить категории    |

## Настройка

### GitHub интеграция

Измените значения в `public/js/github-app.js`:

```javascript
let currentProfile = 'YourGitHubUsername';
const myUsername = 'YourGitHubUsername';
```

### Настройка тем

Темы настраиваются через CSS-переменные в `:root` и `[data-theme="light"]`.

### Лимиты загрузки

- Максимальный размер файла: **500 MB**
- Максимальное количество файлов: **3000**
- Поддерживаемые форматы: **JPEG, PNG, GIF, WebP, SVG, BMP, TIFF, AVIF, HEIF**
